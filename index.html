Diseña CódigoFacilito como si fuera 2011.

Participa en este divertido concurso, envía una página web con HTML, CSS y JavaScript como si fuera la portada de la página de CódigoFacilito diseñada en 2011, año en que lanzamos el proyecto.

Qué calificaremos.

La página debe parecer hecha en el año 2011, puedes usar ésta página para explorar cómo se veían las páginas en ese año.

Sé creativo con temas relacionados a la educación, imagina qué sería revolucionario en 2011 sobre una escuela de programación, diviértete.

Sube tu aplicación a un hosting, no recibiremos carpetas con código, debes enviarons una url, más adelante colocaremos recursos que te ayudarán a plasmar tu idea en una página web.

Cómo participar

  • Crear una página web sobre CódigoFacilito. (Puedes incluir la cantidad de secciones/paginas que desees)
  • Diseñar la página como si fuera 2011.
  • Subir la página a un hosting, el que sea. (Te recomendamos Github o Firebase Hosting, puedes ver instrucciones abajo)
  • Enviar el link de la página a 2011@codigofacilito.com
  • Fecha límite de envío: 31 de Octubre de 2018.
